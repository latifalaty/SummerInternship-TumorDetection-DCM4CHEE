Stage IA – Détection de Tumeurs

Projet Stage d’été (2 mois) : Développement d’outils de détection automatique de tumeurs dans les images médicales DICOM.

Ce projet combine Deep Learning et traitement d’images médicales, en utilisant le serveur DICOM dcm4chee-arc-light pour gérer les données d’imagerie médicale.

🛠️ Compétences et technologies utilisées

Python & Jupyter Notebook – scripts d’analyse et de détection (Detection tumeurs.ipynb, detectiontest.ipynb)

Deep Learning / TensorFlow / Keras – entraînement et utilisation de modèles de classification et segmentation

OpenCV & NumPy – traitement d’images médicales

DICOM / dcm4chee-arc-light – gestion et accès aux données médicales

Git & Git Submodules – intégration du serveur DICOM dans le projet

Analyse des résultats et validation sur des jeux de données médicales réelles.

TensorFlow / Keras – deep learning model training and inference
